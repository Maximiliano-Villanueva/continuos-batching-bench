## How to read the summaries

Each linked file is the `summary.md` export produced by `cb-bench export`. Bullets are generated by rule-based analysis in `evaluation/conclusions.py`:

- **E4** — throughput and p99 vs baseline K=1 for each concurrency level
- **E2** — tail latency split between long and short prompts by wave order
- **E6** — Chat vs Completions API latency delta
- **E7** — throughput and p50 E2E for decode-heavy scenarios (compare OFF vs ON runs in the dashboard for A/B)

Run `make dashboard` with local `results/<run_id>/benchmark.db` files for interactive charts.

```bash
./scripts/serve.sh gemma-4-e4b on  && make bench-gemma-on
make dashboard
```
